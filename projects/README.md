# Projects

Applied/implementation-facing work, as opposed to the pure research write-ups under `research/`.

- `PSNAT_draft_v0.19.1.md` — **[UNTESTED - IN THEORETICAL STAGE]** — Persistent Stateful Neural Architecture'd Transformer: a conceptual design for giving transformers persistent state instead of resetting every session
- `cherdius_spec.md` — **[UNTESTED - IN THEORETICAL STAGE]** — constitutional framework and training spec for an applied instance of Flara's constitutional-AI approach
- `niceturing_spec.md` — **[UNTESTED - IN THEORETICAL STAGE]** — a conversationality benchmark for language models, companion to Cherdius
- `PSNAT-AMDON_draft_v0.1.md` — **[EXPERIMENTAL]** — design spec for distributing/orchestrating PSNAT-style persistent state across a network of API-based models
- `amdon-cli/` — **[EXPERIMENTAL]** — a C# implementation of the AMDON design: a persistent-memory, guard-pipeline agent. Functional, unfinished, not the current focus

None of the theoretical specs (PSNAT, Cherdius, NiceTuring) have an implementation yet — they're included as design documents, not validated results.
